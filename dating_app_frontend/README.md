# Dating App Frontend (React + Vite)

Run:
- `npm install`
- `npm run dev` (http://localhost:5173)

Environment:
- Create `.env` and set `VITE_API_BASE=http://localhost:8000` (or your backend URL)

Pages:
- Login (mock)
- Profile create/edit
- Discover (swipe-like card stack with like/pass buttons)
- Chats (matches and simple polling chat)
- Filters panel in top bar (with presets fetched from backend)
